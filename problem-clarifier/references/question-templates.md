# 追问选项模板

以下模板是常见场景的选项参考，**不是必须执行的脚本**。
遇到模板覆盖不到的情况，根据用户原话自由设计选项。
核心约束不变：可多选（`multiple: true`，`custom: true`）+ 跳过渡底 + 自定义填写。

---

## 产品阶段 → 研究方向对照表

此表在用户的研究对象是产品/服务且存在明确生命周期时参考使用。
如果用户研究对象不是产品（如内部工具、流程），不强行套用。

| 阶段 | 典型研究方向（推荐放前面） | 备注 |
|------|--------------------------|------|
| 创意阶段 | 需求调研、痛点探索、场景挖掘 | 开放性探索为主 |
| 立项阶段 | 体验分析、竞品对标、可行性验证 | 侧重分析论证 |
| 需求阶段 | 需求验证、假设检验、体验自查 | 侧重验证确认 |
| 测试阶段 | 体验测试、可用性测试、需求验证 | 侧重实操反馈 |
| 运营阶段 | 体验评估、用户声音分析、满意度跟踪 | 侧重持续监测 |

**注意**：运营阶段也可能需要做需求调研，创意阶段也可能需要评估现有体验。
阶段只是提供"最常见的做法"作为参考，实际以用户选择为准。此表仅用于排序选项（推荐放前），不做过滤。

---

## 1. 用户提出了需求/痛点类模糊问题

匹配信号："用户还需要什么""用户有什么痛点""用户想要什么功能"

借助 `question` 工具（`multiple: true`，`custom: true`）发送：

```
你提到想了解用户的需求，这通常是以下哪种情况？（可多选）

☐ 想看看现有功能还有哪些体验上的不足，需要改进的地方（如"用户觉得哪里不好用"）
☐ 想挖掘目前还没做、但用户需要的全新功能（如"用户希望新增什么能力"）
☐ 想为下一个版本排功能优先级，看先做哪个（如"几个备选功能，用户最想要哪个"）
☐ 我还不确定，你先帮我建议
☐ 其他（请描述）_______
```

**如果用户已回答所处阶段**，按对照表将推荐方向置顶并标注「推荐」，其余不变。

## 2. 用户提出了体验评估类模糊问题

匹配信号："体验怎么样""好不好用""用户满不满意""用户体验如何"

```
你说想了解用户体验，通常可以从这几个角度切入——（可多选）

☐ 整体满意度，看看用户对产品的总体评价（如"用户给产品打几分"）
☐ 具体功能/流程的易用性，看用户用起来顺不顺（如"用户能不能快速完成XX操作"）
☐ 对某个改版或新功能的效果评估（如"新版本上线后用户反馈变好了还是变差了"）
☐ 我不确定，你先帮我分析一下适合从哪个角度入手
☐ 其他（请描述）_______
```

## 3. 用户提出了行为调研类模糊问题

匹配信号："用户怎么用""使用习惯""使用频率""用户行为"

```
你想了解用户的使用行为，通常涉及这些方面——（可多选）

☐ 使用频率和时长（如"用户多久用一次、每次用多久"）
☐ 使用场景和习惯（如"用户在什么情况下用、怎么用的"）
☐ 核心操作路径和偏好（如"用户完成任务时走什么路径"）
☐ 不同用户群体的行为差异（如"新用户和老用户用起来有什么不同"）
☐ 我还不确定，你先帮我建议
☐ 其他（请描述）_______
```

## 4. 用户提出了态度/感知类模糊问题

匹配信号："用户怎么看""用户觉得""用户认知""品牌感受"

```
你想了解用户的态度和看法，可能是以下方向——（可多选）

☐ 对产品/品牌的整体印象和信任度（如"用户认为这个产品可靠吗"）
☐ 对某个具体功能的价值感知（如"用户觉得这个功能有用吗"）
☐ 与竞品的对比感知（如"用户觉得我们和XX相比怎么样"）
☐ 我不确定，你先帮我建议
☐ 其他（请描述）_______
```

## 5. 用户完全没有方向，只是说"帮我调研一下"

匹配信号："帮我调研""帮我做用户研究""帮我分析一下这个产品"

```
好的，在开始之前我需要先帮你明确一下方向。通常用户研究会围绕这几个目标——（可多选）

☐ 发现问题：了解产品目前有什么体验上的不足或痛点
☐ 验证假设：确认某个想法或改版方向是否正确
☐ 排序决策：在多个功能或方案中确定优先级
☐ 评估效果：看某个版本或功能的实际表现如何
☐ 其他（请简单描述你的想法）_______

如果不确定，可以先选第一个，这是最常见的研究起点。
```

## 补充追问原则

- 所有选项使用 ☐ 前缀，明确可多选；设置 `multiple: true`，`custom: true`
- 如果用户选"不确定/帮我建议"，根据问题的上下文给出最合理的方向并简要说明理由
- 选项中的括号示例必须贴近用户原话的场景，不要用模板里的死例子
- 用户选"其他"时，一句话描述即可，不追问"能再具体点吗"
- 阶段对照表仅用于排序（推荐放前），不做过滤
- 每一轮都包含"跳过"和"其他"兜底选项
