# 样本量计算器 (Sample Size Calculator)

## 使用说明

当用户在需求中明确提供了以下参数时，使用精确公式计算样本量；否则使用各方法参考文件中的经验值。

## 定量研究样本量公式

### 简单随机抽样（无限总体）

**公式**：n = Z² × p × (1-p) / e²

- **Z**：置信水平对应的 Z 值
- **p**：预期比例（未知时取 0.5，最大化样本量）
- **e**：可接受的误差范围（如 0.05 表示 ±5%）

### Z 值速查表

| 置信水平 | Z 值 |
|---------|------|
| 80% | 1.282 |
| 85% | 1.440 |
| 90% | 1.645 |
| 95% | 1.960 |
| 99% | 2.576 |

### 常见组合速查表（p=0.5，无限总体）

| 置信水平 | 误差 ±3% | 误差 ±5% | 误差 ±7% | 误差 ±10% |
|---------|---------|---------|---------|----------|
| 90% | 752 | 271 | 138 | 68 |
| 95% | 1,067 | 384 | 196 | 96 |
| 99% | 1,843 | 663 | 339 | 166 |

### 有限总体校正

当总体规模 N 已知时：

**公式**：n_adj = n / (1 + (n - 1) / N)

其中 n 是无限总体公式计算结果，n_adj 是校正后的样本量。

**例**：无限总体 n=384，总体 N=5000
- n_adj = 384 / (1 + 383/5000) = 384 / 1.0766 ≈ 357

### 有限总体速查表（95% 置信，5% 误差，p=0.5）

| 总体规模 N | 所需样本量 n_adj |
|-----------|----------------|
| 500 | 218 |
| 1,000 | 278 |
| 2,000 | 322 |
| 5,000 | 357 |
| 10,000 | 370 |
| 100,000 | 383 |
| >1,000,000 | ≈ 384 |

## 分层抽样样本量

当需要分层分析时，样本量需按层分配：

### 比例分配
各层样本量 n_h = n × (N_h / N)，即按各层在总体中的比例分配。

### 等额分配
各层样本量相同：n_h = n / H（H 为层数），适用于层间样本量差异不大时。

### 最优分配（考虑层内方差和成本）
n_h = n × (N_h × σ_h / √c_h) / Σ(N_h × σ_h / √c_h)，其中 σ_h 为层内标准差，c_h 为层内单位成本。

**实操建议**：
- 默认使用比例分配
- 若某层样本量 < 30，考虑等额分配补足该层
- 若某层分析需求特别高，可超额分配

## 最小子群要求

- 独立子群分析：每个子群至少 30 人
- 交叉分析单元格：每个单元格期望频数 ≥ 5
- A/B 对比检验：每组至少 50 人

## 定性研究样本量原则

定性研究不适用统计学公式，采用以下原则：

1. **信息饱和原则 (Saturation)**：当连续 2-3 个被访者不再贡献新信息时，即可停止
2. **信息效力原则 (Information Power)**：研究目标越窄、被访者越特定、理论框架越强、访谈质量越高 → 所需样本量越少
3. **实践建议**：
   - 探索性访谈：8-15 人
   - 特定群体聚焦：6-10 人
   - 多群体对比：每组 5-8 人

## 可用性测试样本量（Nielsen 模型）

基于问题发现率的数学模型：

P(discovery) = 1 - (1 - λ)^n

- λ：单个用户发现特定问题的概率（通常 λ ≈ 0.31）
- n：测试用户数

| 测试用户数 | 预期问题发现率 |
|-----------|-------------|
| 3 人 | 67% |
| 5 人 | 85% |
| 8 人 | 95% |
| 10 人 | 97% |
| 15 人 | 99% |

Nielsen 建议：5 人一轮，多轮迭代更有效。

## 计算过程输出格式

当执行精确计算时，在方案中注明计算过程：

```
**样本量计算过程**：
- 置信水平：95%（Z=1.96）
- 预期比例：50%（p=0.5，最大化样本量）
- 可接受误差：±5%（e=0.05）
- 总体规模：未知（按无限总体处理）
- 公式：n = 1.96² × 0.5 × 0.5 / 0.05² = 384
- 建议有效样本量：≥ 384 份
- 考虑 10% 无效问卷：需回收 ≥ 427 份
```
